🏠 EcoSmart Home Infrastructure

🚀 Overview

This repository contains the full cloud infrastructure setup for a Smart Home Management System. The project uses Docker Containerization to orchestrate multiple services, ensuring a scalable and isolated environment for home automation data and user interfaces.

🏗️ Architecture & Services
The ecosystem is built using Docker Compose and consists of:

-Appsmith (UI): A low-code platform for the main user dashboard.

-PostgreSQL (Database): Robust relational database for storing device logs, user data, and system configurations.

-Adminer: A lightweight database management tool for real-time SQL monitoring.

-Mock APIs (Users & Devices): Simulated REST endpoints (custom & pre-built) to demonstrate data fetching and integration.

🛠️ Tech Stack
Orchestration: Docker, Docker Compose

-Database: PostgreSQL 15

-Backend: Node.js / Mock APIs

-Frontend: Appsmith

-Configuration: YAML, SQL, JSON
