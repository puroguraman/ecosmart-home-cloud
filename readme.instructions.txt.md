================================================================

ΟΔΗΓΟΣ ΕΓΚΑΤΑΣΤΑΣΗΣ \& ΕΠΙΔΕΙΞΗΣ (VS CODE DEMO SCRIPT)

Εργασία: Ανάπτυξη Εφαρμογής Microservices "EcoSmart Home"

================================================================



Απαραίτητα Εργαλεία:

1\. Docker Desktop (Να τρέχει στο παρασκήνιο)

2\. Visual Studio Code (VS Code)

3\. Docker Extension για το VS Code (Προαιρετικό, αλλά διευκολύνει)



----------------------------------------------------------------

**ΒΗΜΑ 1: ΑΝΟΙΓΜΑ \& ΕΚΚΙΝΗΣΗ (START-UP)**

----------------------------------------------------------------

1\. Ξε-zipάρετε τον φάκελο της εργασίας.

2\. Ανοίξτε το VS Code.

3\. Πηγαίνετε: File > Open Folder... και επιλέξτε τον φάκελο της εργασίας.

4\. Στην μπάρα αριστερά (Explorer), εντοπίστε το αρχείο "compose.yaml".

5\. Κάντε ΔΕΞΙ ΚΛΙΚ πάνω στο όνομα του αρχείου "compose.yaml".

6\. Επιλέξτε "Compose Up" (ή ανοίξτε Terminal με Ctrl + ` και γράψτε: docker compose up -d).

7\. Περιμένετε μέχρι να δείτε και τα 5 containers να τρέχουν (πράσινο εικονίδιο στο Docker Desktop).



----------------------------------------------------------------

**ΒΗΜΑ 2: ΑΡΧΙΚΟΠΟΙΗΣΗ ΔΕΔΟΜΕΝΩΝ (SETUP)**

----------------------------------------------------------------

Σημείωση: Επειδή τα δεδομένα (Volumes) δεν περιλαμβάνονται στο zip για λόγους χωρητικότητας,

απαιτείται μια σύντομη διαδικασία φόρτωσης (Import) κατά την πρώτη εκτέλεση.



A. Ρύθμιση Frontend (Appsmith):

&nbsp;  1. Ανοίξτε τον Browser στο: http://localhost

&nbsp;  2. Κάντε Login/Sign up (αν ζητηθεί, η δημιουργία λογαριασμού είναι δωρεάν \& τοπική).

&nbsp;  3. Στην αρχική οθόνη, επιλέξτε "Import from File".

&nbsp;  4. Επιλέξτε το αρχείο "EcoSmart Home.json" που υπάρχει στον φάκελο.



B. Ρύθμιση Βάσης Δεδομένων (PostgreSQL):

&nbsp;  1. Ανοίξτε νέα καρτέλα στο: http://localhost:8080 (Adminer)

&nbsp;  2. Συνδεθείτε με τα εξής στοιχεία:

&nbsp;     - Σύστημα: PostgreSQL

&nbsp;     - Διακομιστής (Server): eco-db

&nbsp;     - Όνομα Χρήστη: admin

&nbsp;     - Κωδικός: adminpass

&nbsp;     - Βάση Δεδομένων: ecohome\_db

&nbsp;  3. Πατήστε "Εισαγωγή" (Import) στο μενού αριστερά.

&nbsp;  4. Πατήστε "Επιλογή Αρχείου", διαλέξτε το αρχείο "ecohome\_db" (ή ecohome\_db.sql) που υπάρχει μέσα στον φάκελο της εργασίας.

&nbsp;  5. Πατήστε "Εκτέλεση" (Execute). Θα εμφανιστεί μήνυμα επιτυχίας.



----------------------------------------------------------------

**ΒΗΜΑ 3: ΣΕΝΑΡΙΟ ΕΛΕΓΧΟΥ (TESTING DEMO)**

----------------------------------------------------------------

Το σύστημα είναι πλέον πλήρως λειτουργικό. Ακολουθήστε τα βήματα για έλεγχο:



1\. Επισκόπηση (Read):

&nbsp;  Πηγαίνετε στο Appsmith (http://localhost) και ανοίξτε την εφαρμογή "EcoSmart Home".

&nbsp;  --> Ελέγξτε ότι ο πίνακας εμφανίζει λίστα συσκευών (Device API check).

&nbsp;  --> Ελέγξτε ότι εμφανίζεται η Θερμοκρασία στο πάνω μέρος (Weather API check).



2\. Δοκιμή Εγγραφής (Write Operation):

&nbsp;  --> Συμπληρώστε τη φόρμα κάτω από τον πίνακα (Περιγραφή/Κόστος).

&nbsp;  --> Πατήστε "Αποθήκευση".



3\. Δοκιμή Λογικής (Logic):

&nbsp;  --> Πατήστε την Καρδιά (❤️) σε μια συσκευή του πίνακα.

&nbsp;  --> Πατήστε τη Σπασμένη Καρδιά (💔) σε μια άλλη για αφαίρεση.



4\. Επιβεβαίωση Δεδομένων (Persistence Check):

&nbsp;  --> Πηγαίνετε στο Adminer (http://localhost:8080).

&nbsp;  --> Ελέγξτε τον πίνακα "maintenance\_logs" (Select data) -> Θα δείτε την εγγραφή σας.

&nbsp;  --> Ελέγξτε τον πίνακα "favorite\_devices" (Select data) -> Θα δείτε την αγαπημένη συσκευή.



----------------------------------------------------------------

**ΒΗΜΑ 4: ΤΕΡΜΑΤΙΣΜΟΣ (CLEANUP)**

----------------------------------------------------------------

1\. Επιστρέψτε στο VS Code.

2\. Κάντε ΔΕΞΙ ΚΛΙΚ στο αρχείο "compose.yaml".

3\. Επιλέξτε "Compose Down" (ή στο Terminal: docker compose down).
